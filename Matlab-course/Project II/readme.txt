The package consists of three documents

1. gameCrossLine - template.m

   An unfinished MATLAB file for gameCrossLine Class. The code in this file is the same as the template code given in MATLAB grader.

2. gameCrossLine Test Problems.txt

   This file includes 4 test problems and their solutions.

3. MATLAB_Grader_Description_Snapshot.docx

   This file includes the snapshot of the MATLAB Grader description of this project.

4. ticTacToeGeneralized.m

   A MATLAB document for ticTacToeGeneralized Class, which is similar to this project.
